# объектно-ориентированная модель библиотечной системы
# описание
Проект представляет собой курсовую работу по дисциплине «Объектно-ориентированное программирование» и посвящён разработке программной системы для учёта книг и читателей библиотеки. На выходе реализовано десктопное приложение с графическим интерфейсом на JavaFX. Система позволяет вести электронный каталог книг и журналов, регистрировать читателей, отслеживать выдачу и возврат, а также управлять статусом библиотечных единиц и вести учёт штрафных баллов за просрочку возврата.
# Как начать работу с проектом?
Для начала работы с системой учёта библиотеки пользователю необходимо:
• Склонировать репозиторий с исходным кодом проекта   git clone <https://github.com/Ekaterina01122002/---.git>
• Открыть проект в любой современной Java IDE (например, IntelliJ IDEA), убедившись в наличии установленного JDK версии 17 или выше.
• Собрать проект через систему сборки Maven.
• Запустить приложение с помощью команды: mvn javafx:run
• После старта приложения откроется графический интерфейс, где доступны вкладки для работы с книгами, журналами, читателями и операциями выдачи/возврата.
