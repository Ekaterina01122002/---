# объектно-ориентированная модель библиотечной системы
# описание
Проект представляет собой курсовую работу по дисциплине «Объектно-ориентированное программирование» и посвящён разработке программной системы для учёта книг и читателей библиотеки. На выходе реализовано десктопное приложение с графическим интерфейсом на JavaFX. Система позволяет вести электронный каталог книг и журналов, регистрировать читателей, отслеживать выдачу и возврат, а также управлять статусом библиотечных единиц и вести учёт штрафных баллов за просрочку возврата.
